ATM Transaction System (Python)

📌 Project Description

This is a simple **ATM Transaction System built using Python**.
The program simulates basic ATM operations such as **user authentication, withdrawing money, depositing money, and checking account balance**.

The system stores account details in a Python dictionary and allows users to perform transactions after entering the correct **account number and PIN**.

⚙️ Features

* User authentication using **Account Number and PIN**
* **Withdraw money** from the account
* **Deposit money** into the account
* **Check account balance**
* **Menu-driven ATM interface**
* Handles **invalid login and insufficient balance**

🛠️ Technologies Used

* **Python**
* Basic Python concepts:

  * Dictionaries
  * Conditional statements
  * Loops
  * User input

▶️ How to Run the Project

1. Install Python on your system.
2. Save the code in a file named:

3. Run the program using:

4. Enter the **account number and PIN** to access the ATM menu.
